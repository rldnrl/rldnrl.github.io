Type System

타입 시스템이란?#

타입 검사기가 프로그램에 타입을 할당하는 데 사용하는 규칙을 의미한다.

타입 시스템의 두 가지 종류#

  1. 타입을 명시적으로 알려주는 타입 시스템 (어노테이션을 사용한다.)
  2. 타입을 추론하는 타입 시스템 (어노테이션을 사용하지 않으면 알아서 추론한다.)

1번의 예

let age: number = 100let firstName: string = 'James'

2번의 예

let age = 100let firstName = 'James'

2. 명령형과 선언형

함수형은 선언형 프로그래밍이라는 더 넓은 프로그래밍 패러다임의 한 가지이다. 선언적 프로그래밍은 필요한 것을 달성하는 과정을 하나하나 기술하는 것보다 필요한 것이 어떤 것인지를 기술하는 것에 중점을 두고 애플리케이션의 구조를 세워나간다. 반면, 명령형 프로그래밍은 코드를 원하는 결과를 달성해 나가는 과정에만 관심을 둔다.


예제1#

어떤 문자열을 URL에서 사용할 수 있도록 공백을 -으로 만드는 작업이다.

명령형#

const title = 'class and interface'const urlFriendly = ''
for (let i = 0; i < string.length; i++) {  if (title[i] === ' ') {    urlFriendly += '-'  } else {    urlFriendly += title[i]  }}
console.log(urlFriendly)

선언적 프로그래밍#

const title = 'class and interface'const urlFriendly = title.replace(/ /g, '-')
console.log(urlFriendly)

예제2#

DOM을 만드는 과정이다

명령형#

const target = document.querySelector('#target')const wrapper = document.createElement('div')const headline = document.createElement('h1')
wrapper.id = 'welcome'headline.innerText = 'Hello World'
wrapper.appendChild(headline)target.appendChild(wrapper)

선언형#

const { render } = ReactDOM
const Welcome = () => (  <div id="welcome">    <h1>Hello World</h1>  </div>)
render(<Welcome />, document.querySelector('#target'))

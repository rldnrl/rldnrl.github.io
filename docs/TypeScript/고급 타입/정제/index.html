5. 정제

타입스크립트는 심벌 수행의 일종인 흐름 기반 타입 추론을 수행한다. 즉, 타입 검사기는 typeof, instanceof, in 등의 타입 질의뿐 아니라 마치 개발자가 코드를 읽듯이 if, ?, ||, switch 같은 제어 흐름 문장까지 고려하여 타입을 정제한다.

// 문자열 리터럴 유니온으로 CSS 단위가 가질 수 있는 값을 설명한다.type Unit = 'cm' | 'px' | '%'
// 지원하는 단위를 나열한다.const units: Unit[] = ['cm', 'px', '%']
// 지원하는 단위 각각을 확인하고 일치하는 값이 없으면 null을 반환한다.function parseUnit(value: string): Unit | null {  for (let i = 0; i < units.length; i++) {    if (value.endsWith(units[i])) {      return units[i]    }  }  return null}
type Width = {  unit: Unit  value: number}
function parseWidth(width: number | string | null | undefined): Width | null {  if (width == null) return null; // 1
  if (typeof width === "number") { // 2     return { unit: "px", value: width };  }
  const unit = parseUnit(width);  if (unit) { // 3    return { unit, value: parseFloat(width) };  }
  return null}
  1. 타입스크립트는 영리해서 자바스크립트에서 null에 느슨한 동질성 확인을 수행(width == null)하면 null과 undefined 모두 true를 반환한다.
  2. 런타임에 어떤 값의 타입을 확인할 때 typeof 검사를 사용하는데, 타입스크립트는 컴파일 타입에도 typeof를 사용한다.
  3. parseUnit은 null을 반환할 수 있으므로, 정말 null을 반환하는지 알아보기 위해서 결과가 참인지 확인한다. 타입스크립트는 unit이 참이라면 이 if문에 의해 Unit 타입임이 확실해지고 거짓이라면 null 타입으로 판명된다.

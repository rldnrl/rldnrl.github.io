4. 함수 호출 시그니처

함수 호출 시그니처#

함수 호출 시그니쳐는 타입 수준의 코드, 즉 값이 아닌 타입의 정보만 포함한다. 함수 호출 시그니처로 매개변수 타입, this 타입, 반환 타입, 나머지 타입, 조건부 타입을 표현할 수 있지만 기본값을 표현할 수 없다. 함수 호출 시그니처는 바디를 포함하지 않아 타입스크립트가 타입을 추론할 수 없으므로 반환 타입을 명시해야한다.

타입 별칭을 이용해서 호출 시그니처를 만들 수 있다.

type Greet = (name: string) => stringtype Log = (message: string, userId?: string) => voidtype SumVariadicSafe = (...numbers: number[]) => number

호출 시그니처와 구현의 관계를 더 구체적으로 확인하면 다음과 같다.

type Log = (message: string, userId?: string) => void
const log: Log = (message, useId = 'Not signed in') => {    let time = new Date().toLocaleTimeString()    console.log(time, message, userId)}

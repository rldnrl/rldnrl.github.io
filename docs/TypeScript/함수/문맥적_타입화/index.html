5. 문맥적 타입화

문맥적 타입화#

함수의 매개변수 타입을 명시하지 않아도 되는 경우이다. 

type Log = (message: string, userId?: string) => void
const log: Log = (message, useId = 'Not signed in') => {    let time = new Date().toLocaleTimeString()    console.log(time, message, userId)}

log의 타입을 Log로 지정했으므로 타입스크립트가 message의 타입을 string으로 추론할 수 있었다. 이는 문맥적 타입화라는 타입스크립트의 강력한 타입 추론 기능이다.


다음은 콜백 함수의 예이다.

function times(    f: (index: number) => void,    n: number) {    for (let i = 0; i < n; i++) {        f(i)    }}

times를 호출할 때 함수 선언을 인라인으로 제공하면 인수로 전달하는 함수의 타입을 명시할 필요가 없다.

times(n => console.log(n), 4)

f를 인라인으로 선언하지 않으면 타입스크립트는 타입을 추론할 수 없다.

function f(n) { // error. 매개변수 n의 타입은 암묵적으로 any가 된다.    console.log(n)}
times(f, 4)

객체에서 in 연산자

in 연산자로 프로퍼티(객체의 키) 존재 여부 확인하기#

자바스크립트에서 존재하지 않는 프로퍼티에 대해서 접근할 때, undefined를 반환한다.

const fruits = { }fruits.apple // undefinedfruits.apple === undefined // true

이렇게 undefined를 비교해서 프로퍼티의 존재 여부를 확인할 수 있지만 in을 사용할 수도 있다.

const fruits = { apple: 1, banana: 5 }
"apple" in fruits // true"banana" in fruits // true"mango" in fruits // false

for ... in 반복문#

for ... in 반복문을 사용하게 되면 객체의 모든 프로퍼티를 순회할 수 있다.

const fruits = { apple: 1, banana: 5 }
for (const key in fruits) {  console.log(key)}

tip

for ... in에서 프로퍼티가 출력되는 방식
정수형 프로퍼티는 정렬돼서 출력되고, 그 외에는 입력한 순서대로 출력이 된다.

// 정수형 프로퍼티const codes = {  "50": "javascript",  "49": "typescript",  "48": "python",  "1": "react"}
for (const key in codes) {  console.log(key) // 1, 48, 49, 50}

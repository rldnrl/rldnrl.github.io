1. React 관련 명령어 모음

Project Setup#

Getting Started with TypeScript#

yarn create react-app <project name> --template typescript

Getting Started with RTK + TypeScript#

yarn create react-app <project name> --template redux-typescript

Getting Started with Next.js + TypeScript#

yarn create next-app <project name> --typescript

Getting Started with Next.js + Example#

yarn create next-app <project name> --example <example 이름>

Library Install#

Install React-Query#

yarn add react-query

Install Zustand#

yarn add zustand

Install Emotion#

yarn add @emotion/styled @emotion/css @emotion/react
